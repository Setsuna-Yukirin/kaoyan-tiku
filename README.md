## 上传规范

### 错题记录

上传错题时，请遵守以下规则：

1. **只保留已知信息** —— 题目原文、选项（如有）
2. **不要推测答案** —— 不确定的答案不要写
3. **不要添加解析** —— 解析可能不正确，留给使用者自己思考
4. **标注来源** —— 如年份、统考真题、习题册名称等（如果知道的话）

**示例**：
```markdown
**03.** 链式存储设计时，结点内的存储单元地址（）。
- A. 一定连续
- B. 一定不连续
- C. 可能连续，也可能不连续
- D. 以上都不对

**答案**：A
```
