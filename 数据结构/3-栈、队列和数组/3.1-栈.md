### 第 23 题 - 栈的出栈序列 + C 语言标识符

**题目**：输入序列为 `n1_`（3 个字符依次入栈），输出长度为 3，问有多少种出栈序列可以用作 C 语言标识符？

**你的答案**：A (4 种)  
**正确答案**：C (3 种)  
**错误原因**：5-1=4，只排除了数字开头的 2 种，但实际答案应该是 3 种

**解析**：
- 3 个元素的栈，可能的出栈序列有 **5 种**（卡特兰数 C(3,2)/(3+1) = 5）
- 分别是：`n1_`, `n_1`, `1n_`, `1_n`, `_n1`, `_1n`
- C 语言标识符规则：
  - 只能包含字母、数字、下划线
  - **不能以数字开头**
  - 长度任意
- 排除数字开头的：`1n_`, `1_n` → 剩余 4 种
- 但答案是 3 种，说明还有 1 种被排除（待确认具体原因）

**待确认**：是否有其他限制条件？比如 `_1n` 是否合法？
