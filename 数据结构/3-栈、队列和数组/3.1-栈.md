## 题目

### 综合应用题

**01.** 有 5 个元素，其入栈次序为 A, B, C, D, E，在各种可能的出栈次序中，第一个出栈元素为 C 且第二个出栈元素为 D 的出栈序列有哪几个？

**02.** 若元素的入栈序列为 A, B, C, D, E，运用栈操作，能否得到出栈序列 B, C, A, E, D 和 D, B, A, C, E？为什么？

**03.** 栈的初态和终态均为空，以 I 和 O 分别表示入栈和出栈，则出入栈的操作序列可表示为由 I 和 O 组成的序列，可以操作的序列称为合法序列，否则称为非法序列。
1）下面所示的序列中哪些是合法的？
- A. IOIIOIOO
- B. IOOIOIIO
- C. IIIOIOIO
- D. IIIOOIOO

2）通过对 1）的分析，写出一个算法，判定所给的操作序列是否合法。若合法，返回 true，否则返回 false（假定被判定的操作序列已存入一维数组中）。

**04.** 设单链表的表头指针为 L，结点结构由 data 和 next 两个域构成，其中 data 域为字符型。设计算法判断该链表的全部 n 个字符是否中心对称。例如，xyx、xyyx 都是中心对称的。

**05.** 设有两个栈 S1、S2 都采用顺序栈方式，并共享一个存储区 [0, ..., maxsize-1]，为了尽量利用空间，减少溢出的可能，可采用栈顶相向、迎面增长的存储方式。试设计 S1、S2 有关入栈和出栈的操作算法。

## 错题记录

### 第 23 题 - 栈的出栈序列 + C 语言标识符

**题目**：输入序列为 `n1_`（3 个字符依次入栈），输出长度为 3，问有多少种出栈序列可以用作 C 语言标识符？

**你的答案**：A (4 种)  
**正确答案**：C (3 种)  
**错误原因**：5-1=4，只排除了数字开头的 2 种，但实际答案应该是 3 种

**解析**：
- 3 个元素的栈，可能的出栈序列有 **5 种**（卡特兰数 = 5）
- 分别是：
  1. `SSSXXX` → `_1n` ✅ 合法
  2. `SSXSXX` → `1_n` ❌ 数字开头
  3. `SSXXSX` → `1n_` ❌ 数字开头
  4. `SXSSXX` → `n_1` ✅ 合法
  5. `SXSXSX` → `n1_` ✅ 合法
- C 语言标识符规则：
  - 只能包含字母、数字、下划线
  - **不能以数字开头**
- 排除数字开头的 2 种 → **剩余 3 种**

**错误原因**：
- 错误地枚举了 6 种，把不可能的 `_n1` 也算进去了
- `_n1` 是不可能通过栈操作得到的！因为 n 先入栈、1 后入栈，1 一定在 n 上面，1 必须先出栈
